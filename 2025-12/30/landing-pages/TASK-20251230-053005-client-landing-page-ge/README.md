# Nonprofit Landing Page

A stunning Next.js landing page with emotional storytelling, glassmorphism design, and email capture functionality.

## Features

- Modern 2026 design trends (glassmorphism, generous spacing, subtle gradients)
- Mobile-first responsive design
- Email capture form with validation
- Emotional, conversion-focused copywriting
- Pain → Solution → Benefits → Features → Footer structure
- Scroll animations and micro-interactions
- No external dependencies (uses Tailwind CSS)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- **Colors**: Edit the theme colors in `tailwind.config.js`
- **Content**: Modify copy in `pages/index.js`
- **Email API**: Connect your email provider in `pages/api/subscribe.js`

## Deployment

Deploy to Vercel:
```bash
npm run build
```

The site is optimized for Vercel deployment with zero configuration.
