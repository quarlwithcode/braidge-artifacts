# Braidge Artifacts

Generated artifacts from Braidge HQ Discord bot.

## Structure

```
├── documents/           # Manual document storage
├── {year}-{month}/
│   └── {day}/
│       ├── landing-pages/
│       │   └── TASK-{id}/
│       └── proposals/
│           └── TASK-{id}/
```

## Workers

- **landing-page-generator**: Next.js landing pages
- **proposal-writer**: Markdown proposals with PDFs
